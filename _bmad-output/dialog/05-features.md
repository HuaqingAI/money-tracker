## Core Features by Layer

### Layer 1: Zero-Input Engine (免费层)

| # | Feature | Description |
|---|---------|-------------|
| F1 | 账单自动导入 | 支持微信、支付宝账单文件导入；银行短信/通知解析 |
| F2 | AI智能分类 | 自动识别消费类别（餐饮、交通、娱乐等），学习用户习惯 |
| F3 | 基础报表 | 月度收支概览、分类占比、趋势图 |

### Layer 2: Multi-Dimensional Understanding (付费层核心)

| # | Feature | Description | Primary User |
|---|---------|-------------|-------------|
| F4 | 受益人标注 | 消费可标注"为谁花的"（自己、配偶、孩子、家庭共用） | 已婚家庭 |
| F5 | 共享家庭账本 | 夫妻/家庭成员共同查看、各自视角、隐私权限 | 已婚家庭 |
| F6 | 多身份核算 | 个人账 vs 副业账独立拆分，副业利润自动计算 | 副业人群 |
| F7 | 人情账管理 | 按"人"组织的人情网络，历史礼金追溯，回礼建议 | 人情往来用户 |
| F8 | 消费习惯雷达 | 识别高频小额消费模式，量化"拿铁效应" | 城市白领 |

### Layer 3: AI Butler Intelligence (付费层高级)

| # | Feature | Description |
|---|---------|-------------|
| F9 | AI对话 | 自然语言问答财务问题 |
| F10 | 主动洞察 | AI发现异常或趋势时主动提醒 |
| F11 | 优化建议 | 基于消费模式给出可执行的节省方案 |
| F12 | 消费模拟 | "如果每天少喝一杯咖啡，一年省多少？" |
| F13 | 同类对比 | 与相似画像家庭的匿名消费水平对比 |

| User Type | Key Features |
|-----------|-------------|
| 已婚家庭（大强型） | F1-F3 + F4受益人 + F5共享账本 + F10主动洞察 + F13同类对比 |
| 城市白领（小丽型） | F1-F3 + F8消费雷达 + F10主动洞察 + F11优化建议 + F12消费模拟 |
| 副业人群（小美型） | F1-F3 + F6多身份核算 + F9 AI对话 |

## Free vs Paid Boundary

**免费层（获客）：** F1 + F2 + F3 -- 零记账核心体验
**付费层（变现）：** F4-F13 -- 高成本AI功能 + 最有特色的差异化功能

**逻辑：** 用免费的零记账体验圈住用户，用付费的多维理解和AI智能变现。
