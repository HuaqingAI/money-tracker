# Design Tokens — 了然 (Liaoran)

**Status:** 待定义
**Library:** Tamagui v2
**Last Updated:** 2026-04-11

Design tokens 将随组件添加逐步提取和定义。

---

## Token Categories

### Colors
- Primary colors
- Secondary colors
- Semantic colors (success, error, warning, info)
- Neutral colors
- 特殊语义色（人情中性色、省钱绿、趋势橙+青）

### Typography
- Font families
- Font sizes (12/14/16/20/24/32/40pt)
- Font weights
- Line heights
- Letter spacing

### Spacing
- Spacing scale
- Padding values
- Margin values
- Gap values

### Layout
- Breakpoints
- Container widths
- Safe area insets

### Effects
- Shadows
- Border radius
- Transitions
- Animations（信任剧场动效、Hero数字动画等）

---

**Tokens will be populated as components are specified.**
