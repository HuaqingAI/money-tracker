了然 Design System

AI驱动的零记账智能财务管家 — 基于 Tamagui v2 + Lucide Icons

从 30 个 UX 规格页面中提取 15 个高频复用组件，统一视觉语言和交互模式。支持 React Native (Expo) + Next.js 跨平台 token 共享。

v1.0.0 17 Components Tamagui v2 Lucide Icons
Getting Started

Installation

# Core
npm install tamagui @tamagui/config

# Icons
npm install @tamagui/lucide-icons lucide-react-native

Usage

import { Button, Card, Input, Sheet, Tabs } from 'tamagui'
import { Home, Receipt, Bot, User } from '@tamagui/lucide-icons'
import { liaoran } from './theme/liaoran'
