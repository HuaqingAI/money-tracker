## Component Naming Convention

**Format:** `[type]-[number]`

Examples:
- btn-001 (Button)
- inp-001 (Input Field)
- crd-001 (Card)
- nav-001 (Navigation)
- sht-001 (Sheet/Modal)

## Component File Structure

Each component file includes:
- Component ID
- Type and purpose
- Tamagui base component mapping
- Variants (if any)
- States
- Styling/tokens
- Usage tracking (which pages use this component)
