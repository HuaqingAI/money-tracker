# 07.3-优化建议

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 07: 全员的AI管家体验 |
| **Page Number** | 07.3 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 查看AI生成的可执行省钱建议，每条附具体金额和可量化影响。

**Entry Context:** 从洞察Feed点击"查看建议"进入。

**Exit Action:** 点击"模拟效果"进入消费模拟页。

**On-Page Interactions:**
- 查看3条优化建议列表（每条含具体金额）
- 展开建议详情（"每月减少4次外卖 → 省680元"）
- 标记"采纳"/"忽略"
- 查看已采纳建议的执行追踪
