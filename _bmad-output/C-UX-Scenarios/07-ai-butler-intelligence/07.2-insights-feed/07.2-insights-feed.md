# 07.2-洞察推送

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 07: 全员的AI管家体验 |
| **Page Number** | 07.2 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 浏览AI主动生成的本月消费洞察Feed，发现自己没注意到的消费模式。

**Entry Context:** 从AI对话页点击"查看更多洞察"进入，或从推送通知进入。

**Exit Action:** 点击某条洞察的"查看建议"进入优化建议页。

**On-Page Interactions:**
- 浏览洞察卡片列表（按时间排序）
- 查看洞察详情（"你本月外卖频率比上月高了30%"）
- 标记洞察为"有用"/"不相关"（训练AI）
- 分享单条洞察
