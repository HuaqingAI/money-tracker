# 08.2-个人资料/账户

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 08: 全员的账户与设置管理 |
| **Page Number** | 08.2 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 更新个人信息、查看账户状态和订阅情况。

**Entry Context:** 从设置页点击"个人资料"进入。

**Exit Action:** 关闭页面返回设置或Dashboard ✓（场景完成）

**On-Page Interactions:**
- 修改头像/昵称
- 更新手机号
- 查看订阅状态（免费/付费/到期时间）
- 数据导出（下载自己的消费数据）
- 账户注销入口
