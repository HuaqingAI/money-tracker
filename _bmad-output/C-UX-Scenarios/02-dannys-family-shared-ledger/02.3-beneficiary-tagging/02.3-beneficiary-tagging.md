# 02.3-受益人标注

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 02: 大强的家庭共享账本 |
| **Page Number** | 02.3 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 标注近期消费的受益人维度（"为孩子""为家庭""个人"），让家庭账单有更丰富的归属视角。

**Entry Context:** 创建共享账本并发送邀请后进入。

**Exit Action:** 完成标注，返回Dashboard查看共享视图 ✓（场景完成）

**On-Page Interactions:**
- 浏览近期交易列表
- 为每笔交易选择受益人标签（为自己/为配偶/为孩子/为家庭）
- AI智能建议标签（"超市购物 → 为家庭"）
- 批量标注相似交易
