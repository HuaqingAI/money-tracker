# 02.1-订阅/付费

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 02: 大强的家庭共享账本 |
| **Page Number** | 02.1 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 展示付费功能对比，让大强看到家庭共享账本的价值并完成订阅。

**User Situation:** 用了两周免费版，觉得挺好的。周末小芳问"这个月我们到底花了多少"，想到如果小芳也能看到就不用吵了。

**Mental State:**
- **Hope:** 用共同的数字讨论家庭开支，不再各说各话。
- **Worry:** 付费后老婆不愿意用，钱白花了。

**Discovery Method:** Dashboard上看到"邀请家人共享"提示卡片，点击进入。

**Entry Context:** 从Dashboard的"邀请家人共享"提示卡片点击进入。

**Exit Action:** 完成支付，进入共享家庭账本设置页面。

**On-Page Interactions:**
- 切换月付/年付方案对比
- 查看功能详情展开说明
