# 02.2-共享家庭账本

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 02: 大强的家庭共享账本 |
| **Page Number** | 02.2 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 创建家庭共享账本，生成邀请链接发给小芳加入。

**Entry Context:** 完成付费订阅后自动进入。

**Exit Action:** 发送邀请链接给小芳（微信分享），进入受益人标注页。

**On-Page Interactions:**
- 命名家庭账本
- 生成邀请链接/二维码
- 选择分享渠道（微信）
- 查看共享权限说明（各自隐私消费不共享）
- 等待小芳接受邀请的状态
