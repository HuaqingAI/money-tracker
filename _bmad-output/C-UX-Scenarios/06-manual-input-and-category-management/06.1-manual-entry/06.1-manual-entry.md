# 06.1-手动记账

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 06: 全员的手动记账与分类补充 |
| **Page Number** | 06.1 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 通过表单/语音/截图三种模式快速补充一笔AI未捕获的消费。

**User Situation:** 大强在菜市场花了50块现金买菜，没有电子支付记录。回到家想起来，打开APP补一笔。

**Mental State:**
- **Hope:** 3秒内搞定补充记录，不要打断生活节奏。
- **Worry:** 补一笔消费要填一堆字段，还不如不记。

**Discovery Method:** 打开APP，点击Dashboard底部的"+"快速记账按钮。

**Entry Context:** 从Dashboard底部"+"按钮进入。

**Exit Action:** 确认保存，返回Dashboard或进入分类管理。

**On-Page Interactions:**
- 切换输入模式（表单/语音/截图OCR）
- 语音识别结果确认
- AI自动填充分类的修改
