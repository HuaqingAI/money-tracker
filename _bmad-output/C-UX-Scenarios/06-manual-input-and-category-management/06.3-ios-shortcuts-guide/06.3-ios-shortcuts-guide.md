## Overview

**Page Purpose:** 引导 iOS 用户配置 Siri 快捷指令，实现"嘿 Siri，记一笔午饭38块"的语音快速记账体验。这是"零记账"理念的极致延伸 -- 连打开APP都不需要。

**User Situation:** 大强（或任意 iOS 用户）在分类管理或设置页看到"快捷指令"入口，好奇点进来。他听说过 Siri 快捷指令但从没用过，需要一步步教他怎么设置。

**Entry Points:**
- 06.2 Category Management 页面底部入口（仅 iOS 设备显示）
- 08.1 Settings 页面"快捷指令"入口（仅 iOS 设备显示）

**Exit Points:**
- 配置完成 -> 返回来源页面
- 关闭/返回 -> 返回来源页面

### 核心功能点

| 功能 | 说明 |
|------|------|
| **预设快捷指令** | 提供 2-3 个预设的 Shortcuts 模板，用户点击"添加到 Siri"即可安装。避免用户自己配置参数 |
| **分步图文教程** | 3-4 步配图引导，每步配 iOS 系统截图示意。文字口语化，不用技术术语 |
| **一键安装** | 通过 iOS Shortcuts URL Scheme 或 iCloud 分享链接实现一键安装预设指令 |
| **自定义触发词** | 用户可修改默认触发词（"记一笔"可改为"记账"或其他个人偏好） |
| **测试验证** | 安装后提供测试入口，用户可立刻试用并确认功能正常 |

### 预设快捷指令清单

| 指令名 | 默认触发词 | 功能 | 参数 |
|--------|-----------|------|------|
| 语音记账 | "记一笔{描述}{金额}" | 通过 Siri 语音创建一笔消费记录 | 描述(文本) + 金额(数字) |
| 快速记固定消费 | "记通勤打车" | 记录预设金额的固定消费 | 用户自定义金额+分类 |
| 拍照记账 | "拍照记账" | 打开APP的截图OCR模式 | 无 |

## Content Strategy

**语气：** 友好、鼓励、手把手教。像朋友教你用新功能。

**关键文案方向：**
- Hero: "不用打开APP就能记账" -- 直击价值点
- 步骤标题: "Step 1: 点击添加" / "Step 2: 说出触发词" / "Step 3: 试一试" -- 极简
- 测试: "试试说：嘿Siri，记一笔午饭38块" -- 给具体示例

**禁用词：** "URL Scheme""Shortcuts API""自动化"（技术术语）

---

## Key Design Decisions

1. **预设指令优先于教用户自建** -- 绝大多数用户不会自己从零配置 Shortcuts。提供预设模板 + 一键安装是唯一可行的路径
2. **仅 iOS 设备显示** -- Android 没有等价的 Siri Shortcuts 能力。Android 用户不会看到这个入口。Phase 3 可考虑 Google Assistant Routines
3. **图文教程而非视频** -- 教程页面需要快速扫描，图文比视频更高效。每步一张 annotated screenshot + 一句话说明
4. **测试环节必须有** -- 用户配置完不确定是否成功，一个"开始测试"按钮让他立刻试用并获得信心

## Implementation Notes (Phase 2)

- **依赖：** iOS Shortcuts framework (INShortcut) + SiriKit donation
- **最低版本：** iOS 13+（Shortcuts app 内置）
- **分发方式：** iCloud Shortcuts 分享链接（最简单）或 App 内 INVoiceShortcutCenter 注册
- **深度链接：** 快捷指令通过 URL Scheme 或 Universal Link 调用 APP 的记账接口
- **后台执行：** Siri 触发后可在后台完成记账（无需打开 APP UI），通过 Siri response 确认结果
