# 06.2-分类管理

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 06: 全员的手动记账与分类补充 |
| **Page Number** | 06.2 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 查看和调整AI自动分类规则，确保分类准确率满足预期。

**Entry Context:** 从手动记账确认保存后进入，或从设置页进入。

**Exit Action:** 确认分类规则调整，返回Dashboard或进入iOS快捷指令配置。

**On-Page Interactions:**
- 查看AI分类规则列表
- 修改单条分类规则（如"美团 → 餐饮"改为"美团 → 外卖"）
- 创建自定义分类
- 合并/拆分分类
- 查看分类准确率统计
