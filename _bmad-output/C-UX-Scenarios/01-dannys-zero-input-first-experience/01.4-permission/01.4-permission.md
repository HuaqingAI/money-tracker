# 01.4-权限授权页

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 01: 大强的零记账初体验 |
| **Page Number** | 01.4 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 解释为什么需要通知读取权限，用清晰的隐私承诺消除顾虑，完成授权。

**Entry Context:** 注册完成后自动进入。

**Exit Action:** 点击"允许通知读取"，系统弹出权限请求，授权后进入账单导入页。

**On-Page Interactions:**
- 查看"为什么需要这个权限"的展开说明
- 查看隐私承诺（本地优先、加密存储、不上传原始通知）
- "暂时跳过"选项（可稍后在设置中开启）
