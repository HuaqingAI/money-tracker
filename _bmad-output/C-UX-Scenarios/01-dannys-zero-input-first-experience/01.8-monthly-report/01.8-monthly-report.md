# 01.8-月度报表

## Page Metadata

| Property | Value |
|----------|-------|

## Overview

**Page Purpose:** 展示完整的分类月度报表，支持一键分享给小芳 -- 场景的成功时刻。

**Entry Context:** 从Dashboard点击"查看完整报表"进入。

**Exit Action:** 点击"分享"生成报表卡片发给小芳 ✓（场景完成）

**On-Page Interactions:**
- 查看分类占比饼图（餐饮3200、交通1800...）
- 展开每个分类查看明细
- 切换月份对比
- 点击"分享"生成精美报表卡片
- 选择分享渠道（微信/截图保存）
