# 01.5-账单导入

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 01: 大强的零记账初体验 |
| **Page Number** | 01.5 |

## Overview

**Page Purpose:** 引导大强从支付宝导出CSV账单文件并上传到APP。

**Entry Context:** 授权通知权限后自动进入。

**Exit Action:** 上传CSV文件成功，进入导入处理页。

**On-Page Interactions:**
- 选择导入来源（支付宝/微信/银行）
- 查看分步图文教程（如何从支付宝导出CSV）
- 选择本地CSV文件上传
- 上传进度提示
