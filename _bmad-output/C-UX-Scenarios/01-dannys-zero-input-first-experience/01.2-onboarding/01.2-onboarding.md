# 01.2-Onboarding 引导页

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 01: 大强的零记账初体验 |
| **Page Number** | 01.2 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** 通过3页滑动引导建立"零记账"信任：零输入 → 自动分类 → 智能报表。

**Entry Context:** 从 Welcome 页点击"开始"进入。

**Exit Action:** 滑动完成3页引导，点击"立即体验"进入注册页。

**On-Page Interactions:**
- 左右滑动切换引导页
- 底部进度点指示当前位置
- "跳过"按钮直达注册页
