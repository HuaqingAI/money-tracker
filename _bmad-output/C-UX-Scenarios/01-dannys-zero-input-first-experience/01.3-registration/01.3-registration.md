# 01.3-注册/登录

## Page Metadata

| Property | Value |
|----------|-------|

| **Page Number** | 01.3 |

## Overview

**Page Purpose:** 快速完成账户注册，最少摩擦。

**Entry Context:** 从 Onboarding 引导页完成滑动或点击"立即体验"进入。

**Exit Action:** 提交手机号 + 验证码完成注册，进入权限授权页。

**On-Page Interactions:**
- 输入手机号
- 获取并输入短信验证码
- 微信一键登录（备选）
- 查看用户协议和隐私政策
