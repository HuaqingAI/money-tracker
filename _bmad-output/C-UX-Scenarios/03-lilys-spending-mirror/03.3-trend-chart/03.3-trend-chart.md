# 03.3-趋势图表

## Page Metadata

| Property | Value |
|----------|-------|

## Overview

**Page Purpose:** 查看近3个月消费趋势，发现餐饮类消费逐月上升的模式。

**Entry Context:** 从交易详情或Dashboard进入。

**Exit Action:** 点击"查看消费雷达"进入消费习惯雷达页。

**On-Page Interactions:**
- 查看多月趋势折线图
- 按分类切换查看（全部/餐饮/交通/购物）
- 点击某月数据点查看该月明细
- 月度环比变化高亮提示
