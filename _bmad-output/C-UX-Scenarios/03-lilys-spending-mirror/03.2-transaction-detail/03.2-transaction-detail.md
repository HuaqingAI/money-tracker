# 03.2-交易详情

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 03: 小丽的消费真相镜子 |
| **Page Number** | 03.2 |
| **Platform** | Mobile — iOS |

## Overview

**Page Purpose:** 查看单笔交易的AI识别详情，包括品牌、分类、时间和地点。

**Entry Context:** 从交易列表点击某笔奶茶消费进入。

**Exit Action:** 返回交易列表，或点击"查看消费趋势"进入趋势图表页。

**On-Page Interactions:**
- 查看AI自动识别的品牌（瑞幸咖啡）
- 查看分类归属（餐饮-饮品）
- 修改分类（如果AI识别错误）
- 添加备注标签
