# 04.2-人情智能建议

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 04: 大强的人情账管理 |
| **Page Number** | 04.2 |
| **Platform** | Mobile — Android |

## Overview

**Page Purpose:** AI基于历史往来和场景智能建议回礼区间，记录本次随礼 -- 场景的成功时刻。

**Entry Context:** 从人情账管理页点击"获取回礼建议"进入。

**Exit Action:** 确认回礼金额并记录 ✓（场景完成）

**On-Page Interactions:**
- 查看AI建议的回礼区间（600-800）及理由
- 查看同类场景（婚礼）的参考范围
- 选择最终金额（800）
- 确认并记录本次随礼
- 设置提醒（婚礼日期提醒）
