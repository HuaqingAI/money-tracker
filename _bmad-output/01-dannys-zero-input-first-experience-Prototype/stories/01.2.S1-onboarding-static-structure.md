### 组件结构

```tsx
<SafeAreaView style={{ flex: 1, backgroundColor: colors.surfacePage }}>

  {/* 固定层：顶部跳过按钮 */}
  <View id="onboarding-topbar" style={styles.topbar}>
    <Pressable id="onboarding-topbar-skip" onPress={handleSkip}>
      <Text color="$neutral500">跳过</Text>
    </Pressable>
  </View>

  {/* 轮播容器 */}
  <FlatList
    id="onboarding-slides"
    ref={flatListRef}
    data={slides}
    horizontal
    pagingEnabled
    showsHorizontalScrollIndicator={false}
    renderItem={({ item, index }) => <OnboardingSlide slide={item} index={index} />}
    onViewableItemsChanged={onViewableItemsChanged}
  />

  {/* 固定层：进度点 */}
  <View id="onboarding-progress" style={styles.progressRow}>
    {slides.map((_, i) => (
      <View key={i} style={[styles.dot, i === currentIndex && styles.dotActive]} />
    ))}
  </View>

  {/* CTA 区域 */}
  <View id="onboarding-cta" style={styles.ctaZone}>
    {currentIndex < 2 ? (
      <Button id="onboarding-cta-next" variant="outlined" onPress={handleNext}>
        下一步
      </Button>
    ) : (
      <Button id="onboarding-cta-start" onPress={handleStart}>
        立即体验
      </Button>
    )}
  </View>

</SafeAreaView>
```

## Implementation Steps

### Step 1: 创建路由文件
创建 `app/(auth)/onboarding.tsx`，配置 Expo Router。确认 `(auth)` group 的 `_layout.tsx` 存在。

### Step 2: 定义 slides 数据数组
按上述定义创建包含 3 个 Slide 内容的静态数据，引用 SVG 资产。

### Step 3: 实现 OnboardingSlide 组件
单张 Slide：插画区（约 40% 屏高）+ 标题 + 副标题。所有 Object ID 通过 `testID` 属性绑定。

### Step 4: 实现 FlatList 轮播容器
`pagingEnabled` + `horizontal` + `onViewableItemsChanged` 追踪当前页。

### Step 5: 实现顶部固定跳过按钮
`position: absolute` 或 Flex 布局固定在轮播之上，不随滑动移动。

### Step 6: 实现底部进度点 + CTA 区域
进度点根据 `currentIndex` 高亮；CTA 按钮根据 `currentIndex` 切换次要/主要样式。

### Step 7: 三设备尺寸验证
验证 375 / 393 / 428px 宽度下插画等比显示、标题不截断。
