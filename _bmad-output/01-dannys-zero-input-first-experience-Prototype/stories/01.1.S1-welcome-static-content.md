# Story 01.1.S1: Welcome - 页面结构与静态内容

**Page**: 01.1 Welcome
**Section**: 1 of 2

## Goal

搭建 Welcome 全页静态布局，定义基础设计 token（品牌色、字体、间距），实现从品牌标识到 CTA 按钮的完整视觉层级。产品名「了然」，插画区用占位符。

---

## What to Build

### Design Tokens (定义在 HTML `<style>` 中)

```css
:root {
  /* 品牌色 — 沉稳信任系 */
  --color-primary: #1A6B5A;       /* 深青绿 — 主色 */
  --color-primary-dark: #0F4F42;  /* 深青绿暗 — 按钮hover/active */
  --color-primary-light: #E8F5F1; /* 浅青绿 — 背景点缀 */
  --color-bg: #FFFFFF;            /* 页面背景 */
  --color-text-primary: #1A1A2E;  /* 主文字 — 近黑 */
  --color-text-secondary: #6B7280;/* 副文字 — 灰 */
  --color-text-inverse: #FFFFFF;  /* 反色文字 */

  /* 字体 */
  --font-family: -apple-system, "PingFang SC", "Noto Sans CJK SC", sans-serif;

  /* 间距 (4px base) */
  --space-xs: 4px;
  --space-sm: 8px;
  --space-md: 16px;
  --space-lg: 24px;
  --space-xl: 40px;
  --space-2xl: 64px;

  /* 圆角 */
  --radius-md: 12px;
  --radius-full: 9999px;
}
```

### CSS Layout

- 全页 flex column, justify-content: space-between
- 单屏不可滚动 (height: 100dvh, overflow: hidden)
- Safe area padding (env(safe-area-inset-bottom))
- 品牌区顶部 space-xl padding
- CTA区底部 space-xl padding + safe area

## Implementation Steps

### Step 1: 创建 HTML 文件骨架
创建 `01.1-Welcome.html`，包含 viewport meta、design token CSS 变量、基础 reset 样式。

### Step 2: 实现品牌标识区
Logo 用 SVG 占位（深青绿圆形 + "了" 字），产品名「了然」。

### Step 3: 实现插画占位区
16:9 比例容器，浅青绿背景 + 图标占位，传达"自动捕获"概念。

### Step 4: 实现价值主张区
H1 标题 + 副标题，居中对齐，文字层级清晰。

### Step 5: 实现 CTA 按钮
全宽主色按钮，底部固定，safe area 适配。

### Step 6: 三设备视口验证
检查 375px / 393px / 428px 下的布局表现。

## Acceptance Criteria

### Agent-Verifiable (Puppeteer)

| # | Criterion | Element | Expected | How to Verify |
|---|-----------|---------|----------|---------------|
| 1 | Logo 存在且可见 | `#welcome-brand-logo` | visible, 64x64 | getBoundingClientRect |
| 2 | 产品名显示「了然」 | `#welcome-brand-name` | textContent = "了然" | textContent check |
| 3 | 标题文案正确 | `#welcome-value-headline` | "花了多少，自动知道" | textContent check |
| 4 | 副标题文案正确 | `#welcome-value-subheadline` | contains "不用动手记" | textContent check |
| 5 | 按钮文案正确 | `#welcome-cta-button` | "开始体验" | textContent check |
| 6 | 按钮全宽 | `#welcome-cta-button` | width >= 300px (375vp) | getBoundingClientRect |
| 7 | 页面不可滚动 | `#welcome-page` | scrollHeight <= clientHeight | scroll check |
| 8 | 所有 Object ID 存在 | 9 个 ID | all present in DOM | querySelectorAll |

### User-Evaluable (Qualitative)

- [ ] 视觉层级清晰：品牌 → 插画 → 文案 → 按钮
- [ ] 品牌色（深青绿）传达沉稳信任感
- [ ] 文案不含"记账"一词
- [ ] 三种设备宽度下布局合理

---

## Design Notes

- 品牌色 #1A6B5A 深青绿 — 沉稳、信任、专业，避免金融类常见的蓝色同质化
- 禁用词：记账、轻松、简单、智能
- 插画区为占位，Phase 6 资产生成时替换
