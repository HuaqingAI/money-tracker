### Slide 入场动画

每次 Slide 进入视图时，插画区从 `translateY(16)` + `opacity(0)` 动画到最终位置，时长 400ms ease-out。标题延迟 80ms。

```ts
// 使用 Reanimated 2 / Animated API
const animateSlideIn = (index: number) => {
  // 重置动画值
  illustrationAnim[index].setValue({ opacity: 0, translateY: 16 })
  // 启动动画
  Animated.parallel([
    Animated.timing(illustrationAnim[index].opacity, {
      toValue: 1, duration: 400, easing: Easing.out(Easing.ease), useNativeDriver: true
    }),
    Animated.timing(illustrationAnim[index].translateY, {
      toValue: 0, duration: 400, easing: Easing.out(Easing.ease), useNativeDriver: true
    })
  ]).start()
}
```

### 进度点动画

当前进度点从 6px → 8px（`width` + `height` 动画），颜色从 `$neutral200` → `$brand500`，过渡 < 200ms。

### "立即体验"按钮点击反馈

按钮按下时缩放至 0.96，释放恢复，duration: 100ms。

### Reduced Motion 适配

```ts
useEffect(() => {
  AccessibilityInfo.isReduceMotionEnabled().then(enabled => {
    setReduceMotion(enabled)
  })
}, [])

// reducedMotion = true 时：跳过所有动画，直接显示最终状态
```

### 安卓返回键

```ts
useBackHandler(() => {
  if (currentIndex > 0) {
    flatListRef.current?.scrollToIndex({ index: currentIndex - 1, animated: true })
    return true // 消费事件
  }
  router.back() // 返回 01.1 Welcome
  return true
})
```

## Acceptance Criteria

### 功能验证

| # | 项目 | 预期 |
|---|------|------|
| 1 | 左滑前进 | Slide 向左滑动切换到下一张 |
| 2 | 右滑返回 | Slide 向右滑动切换到上一张（Slide 1 右滑不响应或轻微回弹） |
| 3 | 点击"下一步" | 等效左滑，Slide 切换 |
| 4 | 点击"跳过" | 导航到 Registration，防抖生效 |
| 5 | 点击"立即体验" | 导航到 Registration，防抖生效 |
| 6 | 进度点联动 | currentIndex 变更时，对应进度点放大 + 变色 |
| 7 | 安卓返回键 Slide 1 | 返回 01.1 Welcome |
| 8 | 安卓返回键 Slide 2/3 | 返回上一张 Slide |
| 9 | Reduced Motion | `AccessibilityInfo.isReduceMotionEnabled() = true` 时无动画 |
