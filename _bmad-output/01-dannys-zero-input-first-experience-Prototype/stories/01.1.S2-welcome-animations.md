# Story 01.1.S2: Welcome - 入场动画与交互

**Page**: 01.1 Welcome
**Section**: 2 of 2
**Complexity**: Simple
**Estimated Time**: 15 minutes

---

## Goal

在 S1 静态页面基础上叠加渐进式入场动画序列（Logo → 标题 → 副标题 → 按钮），按钮点击缩放反馈，以及 prefers-reduced-motion 适配。

---

## What to Build

### CSS Animations

```css
@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }
@keyframes slideUp { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } }
@keyframes scalePress { 0% { transform: scale(1); } 50% { transform: scale(0.96); } 100% { transform: scale(1); } }
```

### Animation Sequence (ease-out)

| Element | Animation | Delay | Duration |
|---------|-----------|-------|----------|
| `#welcome-brand` | fadeIn | 0s | 0.3s |
| `#welcome-hero` | fadeIn | 0.2s | 0.4s |
| `#welcome-value-headline` | slideUp | 0.4s | 0.4s |
| `#welcome-value-subheadline` | fadeIn | 0.6s | 0.4s |
| `#welcome-cta-button` | fadeIn | 0.8s | 0.4s |

### JavaScript

- Button click: scalePress animation + navigate after 0.2s
- prefers-reduced-motion: skip all animations, show everything immediately

---

## Acceptance Criteria

### Agent-Verifiable

| # | Criterion | Expected |
|---|-----------|----------|
| 1 | 动画 CSS 存在 | @keyframes fadeIn, slideUp, scalePress defined |
| 2 | 初始状态 opacity: 0 | 所有动画元素初始不可见 |
| 3 | reduced-motion 适配 | @media query 取消所有动画 |
| 4 | 按钮点击有反馈 | scalePress animation on click |

### User-Evaluable

- [ ] 动画序列流畅自然，引导视线从上到下
- [ ] 总时长 < 1.2s，不拖沓
- [ ] 按钮点击反馈明确
